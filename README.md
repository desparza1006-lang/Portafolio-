## 🚀 Cómo Ejecutar Localmente

1. Clona el repositorio:
   ```bash
   git clone https://github.com/desparza1006-lang/Portafolio-.git
   ```

2. Navega al directorio:
   ```bash
   cd Portafolio-
   ```

3. Abre el archivo `index.html` en tu navegador:
   ```bash
   # Opción 1: Doble clic en el archivo
   # Opción 2: Usar un servidor local simple
   python -m http.server 8000
   # Luego visita http://localhost:8000
   ```

> **Nota:** Este es un sitio estático HTML/CSS/JavaScript. No requiere instalación de dependencias ni build process.
